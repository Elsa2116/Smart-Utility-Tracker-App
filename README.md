# Smart Utility Tracker

**Smart Utility Tracker** is a Flutter mobile app that helps users **track and pay for electricity and water usage**. The app uses **SQLite (via sqflite)** for local storage, so all data is stored securely on the user’s device.  



##  Features

- User authentication (Sign Up / Login)  
- Track electricity and water usage  
- View usage history  
- Make utility payments  
- Notifications for due payments  
- Local data storage using **sqflite (SQLite)**  
- Flutter Web support (optional)  


## Technologies Used

- **Flutter & Dart** – mobile & web frontend  
- **sqflite (SQLite)** – local database  
- **Provider / Riverpod / setState** – state management (depending on implementation)  
- **VS Code / Android Studio** – development environment  
- **Git / GitHub** – version control
